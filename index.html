ECE 1100 Discovery Project

Ultrasonic Distance Sensor with Visual Feedback

A handheld hardware device integrating ultrasonic sensing, embedded logic, and Bluetooth communication to provide real-time proximity alerts.

Final Circuit Assembly Image: IMG-20251119-WA0004.jpg

Project Overview

I designed a handheld device to measure distance using an ultrasonic sensor. It uses LED feedback (Green/Yellow/Red) to indicate proximity zones (Near: <10", Mid: 10-20", Far: >20"). This system mimics backup sensors found in modern vehicles.

Components

Arduino Uno

Microcontroller

HC-SR04

Ultrasonic Sensor

HC-05 Module

Bluetooth Comm

Logic & Thresholds

The system calculates distance based on the echo time of the sound wave. The code converts cm to inches (distance_in = distance_cm / 2.54) before applying logic:

Safe (Far): > 20 in
Warning (Mid): 10 - 20 in
Danger (Near): < 10 in

Challenges & Solutions

1. Signal Instability

As seen in the video, the LEDs sometimes flicker between Green and Yellow. This is due to sensor noise around the threshold. Lesson: Future iterations require software debouncing or a moving average filter to smooth the data.

2. Bluetooth Connectivity

Connecting the HC-05 module was difficult due to a baud rate mismatch. The module defaults to 38400 in command mode, but my code was using 9600. Debugging required re-configuring the module via AT commands and ensuring the RX/TX pins were not swapped.

Future Work

I plan to build a Python-based Dashboard using the Bluetooth data to visualize distance trends in real-time, upgrading this from a simple alert device to an IoT monitor.
